Camect - Satel Integra API

Configure the Camect API here to make a connection with the Satel Integra ETHM.

Visit https://local.home.camect.com/ and type in the address WITH "https://" and WITHOUT the "/" at the end.

Password is the email prefix by which the Camect is setup. For example service@company.com; the prefix is "service".

Make sure the Satel Integra is set to a static IP address!

The partition name can't include special caracters. For example '.

If you want to block the alerts when a specific zone is open type in the UNIQUE zonename in here.

By default the zone must be closed for the alerts to be working. If you want the zone to be NO instead of NC type in "not". If you leave the field empty it will be NO.
